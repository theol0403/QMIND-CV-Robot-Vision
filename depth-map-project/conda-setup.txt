# QMIND-cv-depthmap

## Setting up the Conda Environment

- To recreate the Conda environment, follow these steps:


- Clone the repository into a  linux system
- enter these commands 
    $ cd QMIND-CV-Robot-Vision/depth-map-project 
    $ wget https://repo.continuum.io/archive/Anaconda3-5.2.0-Linux-x86_64.sh
    $ bash Anaconda3-5.2.0-Linux-x86_64.sh
- install and accept agreement
- reset terminal
- return to repository and run - conda env create -n QMIND -f environment.yml  
- activate your enviroment with - conda activate QMIND
- if it says enviroment is not set up for conda activate, follow instructions and reset terminal then try again

now your should be set up to work on the project
